## Core Features:

- Microgame Hub: Central hub showcasing 10 distinct microgames designed for stress relief.
- Google Sign-In: Firebase authentication for user login with Google accounts.
- XP and Streaks System: Track user experience points and daily streaks to incentivize regular play.
- Data Persistence: Store user XP, streaks, favorite games, and play history using Firestore.
- Bubble Popper: Simple game where users tap bubbles to pop them and trigger satisfying sound effects.
- Daily Challenge: AI generates personalized daily challenges based on user play history. A tool suggests the 'Today's Game'.
- Pixel Paint Transformation: Pixel Paint microgame enhanced with AI tool which transforms doodles into stylized pixel art.
- Theme Toggle: Light and dark theme toggle button accessible from the top-right corner of every page.

## Style Guidelines:

- Primary color: Soft Lavender (#D0B4DE) to evoke calmness and tranquility.
- Background color: Pale Lilac (#E8DAEF) for a gentle and soothing backdrop.
- Accent color: Light Indigo (#A98DCA) to add a touch of sophistication and depth.
- Body and headline font: 'Poppins' (sans-serif) for a clean, modern, and readable interface.
- Use simple, calming icons for microgame cards and menu options.
- Light and dark theme toggle button accessible from the top-right corner of every page.
- Implement soft transitions and subtle animations to create a seamless user experience.
- Dark theme: matte black background with soft neon pastel accents (mint, lavender, sky blue).
- Animate the transition with a short fade or color-blend effect.
